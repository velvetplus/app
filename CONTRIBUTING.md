# Contributing

This public repository tracks Velvet downloads, release notes, installation issues, and support documentation.

Use issues or pull requests for:

- broken download links
- install problems
- release documentation corrections
- platform-specific playback regressions
- support copy that is inaccurate or unclear

Do not submit:

- application source code
- build scripts
- provider names or provider credentials
- secrets, certificates, keys, profiles, or tokens
- compiled builds as git commits

Release binaries are published through GitHub Releases only.
